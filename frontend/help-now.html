When to Call SES Immediately

These situations require immediate professional assistance

Call (617) 564-4800 Now If You See:

  • No cooling/heating in entire facility
  • Burning smell from HVAC system
  • Smoke or sparks visible
  • Water leaking from units
  • Loud grinding or banging noises
  • Breakers tripping repeatedly
  • Temperature above 85°F or below 60°F
  • Multiple zones not responding
  • Member safety concerns

Critical Emergency Scenarios

Step-by-step guidance for urgent situations

Complete HVAC Failure

  • Call SES: (617) 564-4800
  • Check main circuit breakers (don't reset if tripping)
  • Open emergency ticket in portal
  • Notify members if temperature unsafe
  • Contact regional manager

Fire/Smoke/Burning Smell

  • EVACUATE facility if smoke visible
  • Call 911 FIRST for fire/smoke
  • Turn off HVAC main power if SAFE
  • Use fire extinguisher only if trained
  • Then call SES: (617) 564-4800

Water Leak/Flooding

  • Shut off water if accessible
  • Move equipment away from water
  • Call SES: (617) 564-4800
  • Place caution signs/barriers
  • Document with photos

Extreme Temperature

  • Check SES dashboard for alerts
  • Verify thermostat settings
  • Call SES: (617) 564-4800
  • Post member advisories
  • Consider facility closure if extreme

Electrical Issues

  • Turn off affected breakers
  • Call SES: (617) 564-4800
  • Evacuate area if sparking
  • Do NOT attempt repairs
  • Wait for qualified technician

Unusual Noises

  • Grinding/Scraping: Turn off immediately
  • Banging: Note when it occurs
  • Hissing: Check filters and vents
  • Clicking: Normal if occasional
  • Call SES if persistent: (617) 564-4800

Quick Diagnostic Checklist

Before calling SES, gather this information to speed up resolution

Have This Ready When You Call:

  • Location address and site ID
  • Your name and callback number
  • Detailed description of the problem
  • When the issue started
  • Which zones/areas are affected
  • Any unusual noises, smells, or visible issues
  • Current indoor temperature readings
  • Whether breakers have tripped

GM Safety Guidelines

Know what you can safely do yourself vs. when to call SES

GMs CAN Safely Do:

  • Check thermostat settings and adjust
  • Reset breakers (turn OFF first if comfortable)
  • Document unusual noises/conditions
  • Check for obvious airflow blockages
  • Clean or replace air filters (if accessible)
  • Check for visible water leaks
  • Take photos of any issues
  • Monitor temperature readings

MUST Call SES For:

  • Any electrical work on HVAC systems
  • HVAC unit mechanical repairs
  • Part replacements or installations
  • System programming or software changes
  • Refrigerant-related work
  • Control board modifications
  • Compressor or motor work
  • Wiring changes or additions
